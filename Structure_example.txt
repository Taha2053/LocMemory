│   └── metrics.py         ← precision@K, latency logger, benchmark vs baseline
│
├── rl/                    ← Month 3: RL agent for smart memory selection
│   ├── __init__.py
│   ├── env.py             ← Gym-style environment: state = candidates, action = subset pick
│   ├── agent.py           ← PPO agent: learns which memory combo leads to best answers
│   └── reward.py          ← Reward function: response quality signal (LLM-graded or heuristic)
│
├── security/              ← Month 4: privacy wrapper around memory.py
│   ├── __init__.py

│   └── crypto.py          ← AES encryption/decryption for sensitive memories
│
├── dashboard/             ← Locally hosted web UI (React, runs on localhost)
│   ├── App.jsx            ← Root component, routing
│   ├── MemoryList.jsx     ← Lists stored memories with scores + categories
│   ├── MemoryGraph.jsx    ← Month 2+: Obsidian-style graph of memory relationships
│   ├── MetricsPanel.jsx   ← Retrieval stats: precision@K, latency, hit rates
│   └── LogViewer.jsx      ← Session logs viewer with timestamps

├── tests/

│   └── test_e2e.py        ← Full pipeline: user message → memory → LLM → response

├── .env                   ← LOCALMEM_DB_PATH, LOCALMEM_MD_DIR, LOCALMEM_LLM_MODEL
